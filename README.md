# 🚀 Campaign Management System (MERN)

A powerful, full-stack platform built with the MERN stack (MongoDB, Express, React, Node.js) designed to create, manage, and track marketing campaigns.

## 🌟 Key Features
*   **Campaign Lifecycle:** Full CRUD operations for managing marketing campaigns.
*   **Analytics Dashboard:** Visual performance tracking for clicks, ROI, and reach.
*   **Role-Based Access:** Secure authentication for Admins, Managers, and Viewers.
*   **Asset Management:** Centralized storage for campaign images and documents.
*   **Real-time Updates:** Stay synced with team progress and campaign milestones.

## 🛠️ Tech Stack
*   **Frontend:** React.js, Redux Toolkit, Tailwind CSS, Axios.
*   **Backend:** Node.js, Express.js.
*   **Database:** MongoDB with Mongoose ODM.
*   **Auth:** JSON Web Tokens (JWT) & Bcrypt.js.
